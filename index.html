A basic stack based on AWS using AWS Cloudformation.

This repository contains a basic AWS Cloudformation template created with the aim of using it as a teaching material. Useful for understand how, writing sample json files, you can produce an scalable, secure and maintainable stack.

This template is used for the demo that you will find here.
